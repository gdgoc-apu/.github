## Get involved

New to open source? You are in the right place.

1. Read the [Contributing guide](https://github.com/gdgoc-apu/.github/blob/main/CONTRIBUTING.md)
2. Pick a repo and look for issues labelled `good first issue` *(add labels as you grow)*
3. Open a [bug report](https://github.com/gdgoc-apu/.github/issues/new?template=bug_report.yml), [feature request](https://github.com/gdgoc-apu/.github/issues/new?template=feature_request.yml), or [docs improvement](https://github.com/gdgoc-apu/.github/issues/new?template=docs_improvement.yml)
4. Fork, branch, and open a pull request — maintainers will guide you

**Good first contributions:** documentation, bug reports, UI fixes, workshop materials, and tests.
